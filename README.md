# 🧠 Dual N-Back Cognitive Training App

A professional Dual N-Back web application built using React to improve working memory through real-time visual and audio pattern recognition.

---

## 🚀 Features

* 🎮 Dual N-Back gameplay (visual + audio)
* 🔊 Low-latency audio system (Web Audio API)
* 🎯 User-defined trial selection
* 📊 Real-time accuracy tracking
* 📈 Progress graph visualization
* ⏸️ Start / Pause / Resume / Reset controls
* 🔒 Input locking to prevent spam
* ⚡ Smooth performance using requestAnimationFrame

---

## 🛠️ Tech Stack

* React (Hooks)
* JavaScript (ES6)
* Web Audio API
* Recharts
* Vite

---

## 📊 How It Works

* A sequence of visual positions and audio letters is generated
* User matches current stimulus with N steps back
* Each trial tracks performance
* Results are visualized using a graph

---
